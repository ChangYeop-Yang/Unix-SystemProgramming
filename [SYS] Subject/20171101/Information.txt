Chap 5. 
 
1) my_write (5점) 
   - 예제 write0.c를 수정하여 대화 상대가 로그인 된 경우와 로그인 안된 경우에 대해 구분
   - 대화 상대가 로그인 된 경우 로그인 되어 있음을 메세지로 보여주고 대화 시작
   - 대화 상대가 로그인 안된 경우 로그인 되어 있지 않으므로 대화가 안됨을 알려주고 종료
 
2) my_stty (5점)
    - 예제 showtty.c를 참고하여 stty의 설정값을 변경할 수 있는 my_stty를 작성
    - 실행 방법) ./my_stty 속성 설정값
    - 실행 예) ./my_stty ECHO ON  --> echo 기능 turn on
                  ./my_stty ECHO OFF --> echo 기능 turn off
    - 속성값을 설정하면 현재 설정된 상태를 화면에 보여줌
 
3) my_stty 에서 복합 설정 가능하도록 하면 가산점 (3점)
    - 조합 방법은 자유
    - 실행 방법을 반드시 lms 업로드 시 글로 작성하거나, 별도의 텍스트 파일로 알려줄 것
      (설명 없으면 구현하였더라도 가산점 부가 안함) 
 
4) Makefile 작성 (옵션 2점)
    - 3)번 구현 시만 해당
    - my_stty 와 my_stty_adv 버전별로 생성 가능하도록 작성
